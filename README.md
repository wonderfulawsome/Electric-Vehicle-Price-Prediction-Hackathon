# 전기차 가격 예측 해커톤

## 주제
전기차와 관련된 데이터를 활용하여 **전기차 가격을 예측하는 AI 알고리즘 개발**

---

## 설명
- 전기차와 관련된 데이터를 활용하여 전기차 가격을 예측하는 AI 알고리즘을 개발합니다.

---

## Dataset 정보

### `train.csv`
| 컬럼명       | 설명                 |
|--------------|----------------------|
| **ID**       | 샘플별 고유 ID       |
| **제조사**   | 차량 제조사          |
| **모델**     | 차량 모델            |
| **차량상태** | 차량 상태            |
| **배터리용량**| 배터리 용량 (kWh)   |
| **구동방식** | 차량 구동 방식       |
| **주행거리(km)** | 주행 거리 (km)    |
| **보증기간(년)** | 보증 기간 (년)    |
| **사고이력** | 사고 이력 유무      |
| **연식(년)** | 차량 연식 (년)      |
| **가격(백만원)** | 차량 가격 (백만원) |
